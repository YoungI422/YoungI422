<h2>🌱 About Me</h2>
<p>

프론트엔드는 사용자가 직접 마주하는 영역이라고 생각하기 때문에,<br>
단순한 기능 구현을 넘어 사용자의 입장에서 이해하기 쉬운 화면과 흐름을 만드는 것에 관심이 많습니다.<br><br>

현재 JavaScript를 중심으로 웹의 기본 동작 원리를 학습하고 있으며,<br>
HTML, CSS를 공부하고 있고 React 학습을 준비하고 있습니다.<br><br>

아직 프로젝트 경험은 없지만,<br>
기초를 바탕으로 실제로 동작하는 화면을 직접 구현하며<br>
사용자 경험을 개선하는 개발자로 성장하고 싶습니다.
